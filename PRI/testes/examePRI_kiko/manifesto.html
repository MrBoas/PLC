Resumo

No ex1.1 usei o script 'newID.js' que pega no 'movies.json' original dado pelo professor e gera um novo 'new_movies.json' com os ids no formato pedido no enunciado. Depoid foi usado o Mongo e para executar o carregamento deve-se colocar a correr o servidor, usando o script 'mongod.sh' na pasta mongoMovies, e depois executar o script de carregamento 'importMovies.sh' que import o novo dataset ('new_movies.json').

No ex1.2 basta executar 'npm start' na pasta 'myMovies' e o servidor corre na porta default 3000, tendo ja o servidor mongo a correr.

No ex2.1 de aquecimento, foram obtidos os dados pretendidos e colocados na pasta 'aquecimento_respostas'. Esta pasta contem um ficheiro json com os resultados de cada alinea, e um ficheiro texto que contem a resposta a cada alinea.

No ex2.2 basta executar 'npm start' na pasta 'myClav' e o servidor corre na porta default 3000.
